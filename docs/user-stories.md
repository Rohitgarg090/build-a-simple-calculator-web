# User Stories

## Calculator Operations
- **US-001** As a user, I want to click numeric buttons to enter numbers so that I can build an expression.
- **US-002** As a user, I want to click operator buttons (+, −, ×, ÷) so that I can define the operation between two numbers.
- **US-003** As a user, I want to press the equals (=) button so that I can see the result of my expression.
- **US-004** As a user, I want to press the Clear (C) button so that I can reset the calculator and start a new calculation.
- **US-005** As a user, I want to press the Backspace (⌫) button so that I can correct a mistaken digit entry.
- **US-006** As a user, I want to enter decimal numbers using the dot (.) button so that I can perform calculations with non-integer values.
- **US-007** As a user, I want to see an 'Error' message when I divide by zero so that I understand the operation is invalid.
- **US-008** As a user, I want to chain multiple operations (e.g., 5 + 3 × 2) so that I can perform sequential calculations efficiently.

## Display
- **US-009** As a user, I want to see the current expression on the display panel as I type so that I can review what I have entered before calculating.
- **US-010** As a user, I want the result to be displayed prominently after pressing equals so that I can read it easily.

## Keyboard Support
- **US-011** As a user, I want to use my physical keyboard to enter digits and operators so that I can calculate faster without using the mouse.
- **US-012** As a user, I want to press the Enter key on my keyboard as an alternative to the equals button so that my workflow feels natural.

## Calculation History (Optional)
- **US-013** As a user, I want each completed calculation to be automatically saved so that I can refer back to previous results.
- **US-014** As a user, I want to navigate to the /history page so that I can view a list of my past calculations.
- **US-015** As a user, I want to clear my calculation history so that I can remove records I no longer need.

## Responsive & Accessibility
- **US-016** As a user on a mobile device, I want the calculator layout to adapt to my screen size so that all buttons are easy to tap.
- **US-017** As a user relying on assistive technology, I want all buttons to have descriptive ARIA labels so that I can use the calculator with a screen reader.
