# Features

## Core Calculator
- **Numeric Input**: Buttons 0–9 for digit entry
- **Decimal Support**: Dot (.) button for floating-point numbers
- **Arithmetic Operations**: Addition (+), Subtraction (−), Multiplication (×), Division (÷)
- **Equals**: Evaluate and display the result
- **Clear (C)**: Reset the current input and result
- **Backspace (⌫)**: Delete the last entered character
- **Chained Operations**: Support sequential calculations without pressing equals between each step
- **Error Handling**: Display 'Error' for invalid operations (e.g., divide by zero)

## Display Panel
- **Live Expression Display**: Show the current expression as the user types
- **Result Display**: Show the computed result prominently
- **Responsive Layout**: Adapts to mobile and desktop screen sizes

## Optional: Calculation History
- **Save to Supabase**: Persist each completed calculation (expression + result + timestamp)
- **View History**: Display past 20 calculations on the /history page
- **Clear History**: Button to delete all saved history records

## UI/UX
- **Tailwind CSS Styling**: Clean, modern dark or light theme
- **Keyboard Support**: Allow users to type calculations using their physical keyboard
- **Accessible Buttons**: Proper ARIA labels on all interactive elements
